# Vue 3 + TypeScript + Vite

Task: https://github.com/kakchan/vuejs-test?tab=readme-ov-file

SETUP:
`npm create vite@latest <name of app>`
Framework → Vue
Variant → TypeScript

`npm install
npm run dev`

For API and State Management
`npm install pinia axios `

For Testing
`npm install -D vitest @vue/test-utils jsdom`

Styling used is Naive UI
`npm i -D naive-ui`
